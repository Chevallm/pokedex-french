# pokedex-french
***

Example of data:

- Nom (Bulbizarre)
- Name (Bulbasaur)
- Type (plante)
- Type_2 (poison)
- Attaque (49)
- Attaque_special (65)
- Defense (49)
- Defense_special (65)
- Vitesse (45)
